## Как се играе
1. Въвеждаш прякор (без регистрация/парола)
2. **60 секунди** за **10 задачи**
3. Избираш правилния отговор от 4 варианта
4. Събираш точки — бонус за скорост и серии

---

## Точкуване
| | Точки |
|--|-------|
| Верен отговор | +100 |
| Бонус скорост < 3 сек | +60 |
| Бонус скорост < 6 сек | +30 |
| Серия от 3+ верни | +25 на всеки 3 |
